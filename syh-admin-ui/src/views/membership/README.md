# 会员系统前端界面

## 管理后台界面 (syh-admin-ui)

### 已创建页面

#### 1. 会员管理页面 (`/membership/list`)
**文件**: `syh-admin-ui/src/views/membership/MembershipList.vue`

**功能**:
- 会员列表查询（支持按用户ID、等级、状态筛选）
- 会员详情查看
- 会员续费
- 会员升级
- 会员取消
- 分页展示

**特性**:
- 会员等级标签（免费/铜牌/银牌/金牌/白金）
- 会员状态标签（激活/过期/已取消）
- 自动续费状态显示
- 响应式表格布局

#### 2. 会员配置管理页面 (`/membership/config`)
**文件**: `syh-admin-ui/src/views/membership/ConfigManage.vue`

**功能**:
- 会员配置列表展示
- 新增会员配置
- 编辑会员配置
- 启用/禁用配置
- 删除配置

**配置项**:
- 配置名称
- 会员类型（个人/企业）
- 会员等级
- 价格
- 有效期（月）
- 折扣率
- 积分倍率
- 描述
- 启用状态

### API接口
**文件**: `syh-admin-ui/src/api/membership.js`

包含所有会员管理相关的API调用：
- 会员管理API（查询、购买、续费、升级、取消）
- 会员配置API（CRUD操作）
- 会员权益API（查询、分配、消费、检查）

### 路由配置
已在 `syh-admin-ui/src/router/index.js` 中添加：
```javascript
{
  path: 'membership/list',
  name: 'MembershipList',
  component: () => import('@/views/membership/MembershipList.vue'),
  meta: { title: '会员管理', icon: 'Postcard' }
},
{
  path: 'membership/config',
  name: 'MembershipConfig',
  component: () => import('@/views/membership/ConfigManage.vue'),
  meta: { title: '会员配置', icon: 'Setting' }
}
```

## 使用说明

### 管理后台

1. **访问会员管理**:
   - 登录管理后台
   - 点击侧边栏"会员管理"菜单
   - 查看所有会员列表

2. **配置会员等级**:
   - 点击"会员配置"菜单
   - 点击"新增配置"按钮
   - 填写配置信息并保存

3. **管理会员**:
   - 在会员列表中可以查看会员详情
   - 对激活状态的会员可以进行续费、升级、取消操作
   - 使用搜索功能快速定位会员

### C端移动端

1. **查看会员信息**:
   - 访问会员中心页面
   - 查看当前会员等级和权益

2. **开通会员**:
   - 点击"开通会员"按钮
   - 选择想要的会员等级
   - 确认购买

3. **升级会员**:
   - 在会员中心点击"升级会员"
   - 选择更高等级
   - 确认升级
